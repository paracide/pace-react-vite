Unordered Lists

UL stands for "unordered list". This is for lists that are not in any particular order. Lists also use an LI tag. The LI tag stands for "List Item". You can have as many list items as you like, but you MUST have at least one.

Ordered Lists

Ordered lists are for items that need to come in a particular order. Example: A recipe. or driving directions. For these, it's crucial that the items are in the correct order. As always, make sure to choose the right list, logically. Don't use an ordered list if the items don't need to be in order.

Ordered lists use the OL tag

  1. Do this
  2. Then do this
  3. Also, do this
  4. Forgot about this...
  5. Finally, do that!

Description List

These were originally known as "Definition Lists" in previous versions of HTML. They are used to present a term, in the DT or "Description Term" tag, then describe that term in the DD or "Description Detail".
