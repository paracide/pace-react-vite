Using Colors in CSS

CSS Allows Colours to be Specified in Multiple Ways

  1. Named Colours. ie: "darkslategreen. There are many of these colours, but they are not very accurate, and have even known to be inconsistent across different browsers.
  2. RGB Colours. ie: rgb(255,0,0). This is a FULL RED color, no green (0), no blue (0).
  3. RGB Percentage. ie: rgb(100%,0%,0%). ALso red, but using percent instead.
  4. Hexadecimal (long hex). ie: #ffffff. This is known as "Long Hex", using 6 digits for the color. Old HTML coloring could only use long hex.
  5. Short hex. If we are using hex colours with "pairs" of numbers, like #00FF00, we can combine the paired numbers. This is also known as "shorthand". So #00FF00 can be written as #0F0.   6. RGB with Alpha (RGBA) rgba(0,0,255,1). The last number, is ALPHA, or transparency. An ALPHA of 1 is completely opaque. An alpha of 0 is completely transparent. An ALPHA of .5 is going to be 50% transparent.
