Some CSS Length Units

CSS uses 2 types of lengths units: absolute and relative. There are very few times in CSS that you don't need a length unit. (color are an example) Usually, most values require some sort of length unit (or color).

Absolute Length Units

  • Inch -  in 
  • Centimeter -  cm 
  • Millimeter -  mm 
  • Points (1/72") -  pt 
  • Pica (12 points. 6 picas = 1 inch -  pc 
  • Pixels -  px 

Relative Length Units

  • Percent -  % 
  • M-height: Height of capital Letter "M" -  em 
  • x-height: Height of the lowercase letter "x" -  ex 
  • Viewport Width -  vw 
  • Viewport Height -  vh 
  • Viewport Max - vmax
  • Viewport Min - vmin
  • "Root" EM, or REM -  rem 

Testing EX heights

The ex length unit is supposed to be the exact height of the lower-case letter "x". However, some browsers have historically applied this incorrectly. Some browsers treat it as "half of an em", which is not typographically correct. Applied correctly, the "ex" unit should be different for each font.
